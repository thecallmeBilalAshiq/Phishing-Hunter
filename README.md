- Information Security (IS) Project made with the help of Machine/Deep Learning model. Training a GPT-2 model to classify URLs into three categories: benign, defacement, and phishing, using a dataset of labeled URLs. The model is fine-tuned using custom tokenization and special tokens to identify and predict the type of each URL based on its content.
